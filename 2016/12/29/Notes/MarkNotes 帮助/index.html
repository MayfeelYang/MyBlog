MarkNotes 帮助

tags:帮助, 应用

Markdown

Markdown 是一种轻量级标记语言。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML或者格式。
你可以在 这里 找到 Markdown 快速参考

高级用法

引用另外一篇笔记

在笔记中引用另外一篇使用如下语法:


[{label}](marknote://{title})

这里 {label} 就是你想显示的文字，而{title} 则是所引用笔记的标题。注意, 你需要用 %20来替换掉所有在标题中出现的空格。

使用 tag 来组织笔记

使用Tag将你的笔记组织得井井有条。
要在笔记中加入tag，请在标题下添加一行，其格式如下：
Tags:{tag1},{tag2}一篇笔记可以添加一个或者多个tag。

构建静态博客网站

我的博客 就是用这个应用构建的，你相信吗？
照着下面的步骤你也可以轻松构建自己的静态博客!

1. 配置
首先你需要做一些简单配置。 这个配置只需要做一次。
在左边的搜索框中, 输入 settings, 你可以找到一篇名为 settings的笔记。点击选中这篇笔记，找到下面的值，修改成泥需要的配置。

配置项 功能描述
SiteName 你站点的名字
Base URL 你站点的名字的 URL，用于 sitemap。 可选。
SiteDesc 你对自己站点的描述。会显示在页面底部。
github-user 你的github用户名, 可选
twitter-user 你的twitter用户名, 可选
hidden-tags 在这里列入你不想导出的标签，包含其中任一标签的笔记将不被导出。多个标签需要用英文的逗号隔开。
export-tags 如果这个值存在，只有包含这些标签的笔记会被导出。多个标签需要用英文的逗号隔开。

2. 过滤需要导出的笔记
默认情况下, 所有的笔记都会导出。如果只想导出部分笔记的话，你可以在搜索框和Tag 下拉框中输入值来对笔记进行过滤。

3. 导出
现在你可以导出所有的笔记。选择菜单项“文件”—>“将所有笔记导出为静态博客站点"。
快捷键是苹果键+L。将笔记导出到文件夹后你可以发布到任何支持HTTP站点的服务器。

常见问题

1. 为什么生成的静态站点本地在浏览器中可以浏览每一个页面，但是链接导航点击无效？
答: 生成的静态站点有一些javascript，在本地模式下不能工作。你一个Web服务器来运行这个静态站点。
